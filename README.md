# NLPコーチング ガイド

NLPコーチングのコーチ用ガイドアプリと、ガイドブックPDF一式です。

## ファイル構成

| ファイル | 内容 |
|---|---|
| **app.html** | コーチ用ガイドアプリ本体（ダブルクリックで起動・サーバー不要） |
| **guidebook.pdf** | NLPコーチング ガイドブック（全22ページ・19ワーク収録） |
| build_guidebook.py | ガイドブックPDFの生成スクリプト（再生成用） |

## アプリの使い方

`app.html` をダブルクリックしてブラウザで開くだけで使えます。インターネット接続も不要です。

### 4ステップの流れ

1. **① ヒアリング**
   - クライアントに事前に**5〜10個の課題を箇条書き**で持参してもらい、リストに記入
   - その中から**今回取り組む課題を1つ選択**し、現状／望む状態・課題領域・**困りごとの強さ（10段階）**・**セッション終了後にどうなっていたいか（今日のゴール）** を確認
2. **② ワーク提案** — 入力内容から相性の良いワークを3つ自動提案（理由・所要時間・難易度つき）
3. **③ 進行ガイド** — 選んだワークを、コーチの声かけ例・観察ポイント・チェックリスト・タイマーを見ながらリアルタイムで進行
4. **④ 評価** — ワーク後の会話データ（テキスト）をアップロードすると、6つの視点で自動評価

> **残りの課題に取り組む**：時間がある場合は、履歴の各セッション詳細から「📋 残りの課題に取り組む」を押すと、同じクライアントの課題リストを引き継いで別の課題のセッションを始められます（実施済みの課題には「✓ 実施済」が表示されます）。

### 🎙 セッション録音 ＋ 自動文字起こし

画面下の録音バーから、ヒアリング〜ワーク終了まで（画面を移動しても）セッションを録音できます。

- **録音を開始** — マイク許可と同意確認のうえ録音開始。一時停止／再開・経過時間を表示。話者（コーチ／CL）を切り替えると文字起こしに話者ラベルが付きます
- **自動文字起こし** — 録音中に会話をリアルタイムでテキスト化（Chrome / Edge 推奨）
- **終了すると** — 音声を端末内（IndexedDB）に保存し、文字起こしを **④評価** に自動連携。「この文字起こしで評価する」ボタンですぐに採点できます
- **履歴から再生** — 各セッション詳細で録音を再生でき、文字起こしの表示・コピーも可能

**動作要件・留意点**
- 録音は**クライアントの同意のうえ**で行ってください（開始時に確認が出ます）
- マイクの使用許可が必要です。`app.html` をダブルクリック（`file://`）でも Chrome / Firefox なら音声録音は動作します
- **自動文字起こしは Chrome / Edge 中心**です。Safari など非対応ブラウザでは音声録音のみ（文字起こしは手動）。長時間は無音で途切れるため内部で自動再起動しますが、精度は「下書き」レベルとしてご利用ください
- より安定して文字起こしを使うには、ローカルサーバー（例：`python3 -m http.server`）で `http://localhost` から開くのがおすすめです
- 音声・文字起こし・履歴はすべて**端末内にのみ保存**され、外部送信されません

### フィードバックPDFの出力（追加機能）

評価後、2種類のPDF資料を出力できます（印刷ダイアログで「PDFに保存」を選択）。

- **クライアント向け** — 課題整理・取り組んだワーク内容・今日の輝き・次の一歩をまとめた、お渡し用の資料
- **コーチ向け** — 評価スコアの詳細・項目別の所見・次回への重点アクションをまとめたレポート

履歴画面の各セッション詳細からも、同じPDFを後から再出力できます。

### データの保存

- すべてのデータは**お使いのブラウザ内（localStorage）にのみ保存**され、外部には送信されません
- 左メニューの「⬇ 書き出し」でバックアップ、「⬆ 読み込み」で復元・別端末への移行ができます

## ガイドブックPDFの再生成

```bash
python3 build_guidebook.py
```

※ `reportlab` が必要です（`pip install reportlab`）。日本語フォントは内蔵CIDフォントを使用するため、別途インストールは不要です。
